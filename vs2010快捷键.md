##vs2010快捷键
####1.最实用的几条快捷方式
```javascript
/ 在英文状态下将选中内容注释掉
Ctrl-K + Ctrl-C 注释一段选择代码
Ctrl-K + Ctrl-U 取消一段选择代码的注释
Shift+F12 查找所有引用
Ctrl+Shift+A 在项目中添加新项
Shift+Alt+A 在项目中添加已有项
Shift+Alt+O 利用VA快速查找文件
Ctrl+Shift+F 在文件中查找
F4 逐条查看选中内容
Alt+O  hpp文件和cpp文件切换
Ctrl+ U转小写 
Ctrl+ Shift+ U转大写 
F12  Alt+ G 转到定义
Shift+Delete Ctrl+ L 整行删除
```
####2.常用的快捷键
```javascript
Ctrl + Shift + B生成解决方案 
Ctrl + F7 生成编译 
Ctrl + O 打开文件 
Ctrl + Shift + O 打开项目
Ctrl+ J 显示成员列表 
Ctrl+ F12转到声明 
Ctrl+Shift+空格键 显示参数说明
Ctrl+ F4关闭文档窗口 
Ctrl+ A全选 
Ctrl+ W选择当前单词 
F9  打断点 
Ctrl+F9 启用/停止断点 
Ctrl+ Shift+ F9 取消所有断点 
Ctrl+ Shift+ F5调试-重新开始 
F5  运行调试 
Ctrl+F5 开始执行(不调试) 
Shift+F5  停止调试 
Ctrl+ Break取消编译 
F10 跨过程序执行
F11 单步逐句执行
Ctrl+F  查找 
Ctrl+H  替换 
Alt+F12  查找符号(列出所有查找结果)
```
####3.不是很常用的快捷键
```javascript
Ctrl+ Shift+ C显示类视图窗口 
Ctrl+ Shift+ E显示资源视图 
Ctrl+ ALT + J对象浏览 
Ctrl+ ALT + F1帮助目录 
Ctrl+ F1,  F1 帮助 
Shift+ F1当前窗口帮助 
Shift+Alt+Enter 全屏显示
Ctrl+ Page Down光标定位到窗口上方 
Ctrl+ Page Up光标定位到窗口下方 
Ctrl+ F6 , Ctrl+ TAB下一个文档窗口 
Ctrl+ Shift+ F6 , Ctrl+ Shift+ TAB上一个文档窗口 
Ctrl+ M折叠，展开代码定义
Ctrl+ Delete删除至词尾（从词尾删除至光标位置） 
Ctrl+ Backspace删除至词头 （从词头删除至光标位置）
Ctrl+ Shift+ End选择至文档末尾 
Ctrl+ Shift+ Page Down选择至文档末尾开始 
Shift+ End选择至行尾 
Shift+ Page Down选择至行开始处 
Ctrl+ Shift+ Page Up选择至本页前面 
Ctrl+ Shift+ Page Down选择至本页后面 
Ctrl+ End文档定位到最后 
Ctrl+ Page Down文档定位到最前 
Ctrl+ G 转到指定行
F3  查找下一个 
Shift+F3  查找上一个 
Ctrl+Shift+V  剪贴板循环 
Ctrl+左右箭头键  一次可以移动一个单词 
Ctrl+上下箭头键  滚动代码屏幕，但不移动光标位置。
Shift+Alt+箭头键  选择矩形文本 
Alt+鼠标左按钮  选择矩形文本 
Ctrl-M + Ctrl-O / Ctrl-M + Ctrl-P 折叠定义/展开所有代码（停止大纲显示 ）
Ctrl+K,Ctrl+F 格式化代码
Ctrl+Shift+N 新建项目
```
