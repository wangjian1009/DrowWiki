##vs2010快捷键
####1.常用的快捷键
1. Ctrl + Shift + B生成解决方案 
2. Ctrl + F7 生成编译 
3. Ctrl + O 打开文件 
4. Ctrl + Shift + O 打开项目
5. Ctrl+ J 显示成员列表 
6. F12 转到定义 
7. Ctrl+ F12转到声明 
8. Ctrl+Shift+空格键 显示参数说明
9. Ctrl+ F4关闭文档窗口 
10. Shift+Delete Ctrl+ L 整行删除
11. Ctrl+ U转小写 
12. Ctrl+ Shift+ U转大写 
13. Ctrl+ A全选 
14. Ctrl+ W选择当前单词 
15. F9  打断点 
16. Ctrl+F9 启用/停止断点 
17. Ctrl+ Shift+ F9 取消所有断点 
18. Ctrl+ Shift+ F5调试-重新开始 
19. F5  运行调试 
20. Ctrl+F5 开始执行(不调试) 
21. Ctrl+ Break取消编译 
22. F10 跨过程序执行
23. F11 单步逐句执行
24. Ctrl+F  查找 
25. Ctrl+H  替换 
26. Alt+F12  查找符号(列出所有查找结果) 
27. / 在英文状态下将选中内容注释掉
28. Ctrl-K + Ctrl-C 注释一段选择代码
29. Ctrl-K + Ctrl-U 取消一段选择代码的注释
30. Shift+F12 查找所有引用
31. Ctrl+Shift+A 在项目中添加新项
32. Shift+Alt+A 在项目中添加已有项
33. Shift+Alt+O 利用VA快速查找文件
34. Ctrl+Shift+F 在文件中查找
35. F4 逐条查看选中内容
36. Alt+O  hpp文件和cpp文件切换
####2.不是很常用的快捷键
1. Ctrl+ Shift+ C显示类视图窗口 
2. Ctrl+ Shift+ E显示资源视图 
3. Ctrl+ ALT + J对象浏览 
4. Ctrl+ ALT + F1帮助目录 
5. Ctrl+ F1,  F1 帮助 
6. Shift+ F1当前窗口帮助 
7. Shift+Alt+Enter 全屏显示
